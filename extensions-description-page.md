### 1. Shorts Killer - Hide YouTube Shorts

**YouTube Shortsを完全に非表示にして、集中力を取り戻す**

| 項目 | 内容 |
|------|------|
| 価格 | 無料 |
| 状態 | ✅ 公開中 |
| Chrome Web Store | [インストール](https://chromewebstore.google.com/detail/shorts-killer-hide-youtub/bajebpkjihldiceceompbcjgepnbhnni) |

#### 概要
YouTube Shortsは中毒性が高く、気づけば何時間も消費してしまいます。Shorts Killerは、YouTubeからShortsを完全に削除し、本当に見たいコンテンツだけに集中できる環境を提供します。

#### 主な機能
- ホームフィードからShortsを非表示
- サイドバーのShortsタブを削除
- 検索結果からShortsを除外
- 登録チャンネルページからShortsを非表示
- ワンクリックでON/OFF切り替え

#### こんな人におすすめ
- YouTubeで無駄な時間を減らしたい人
- 長尺コンテンツに集中したい人
- デジタルデトックスを実践したい人

**AIプロンプトを保存・管理・即座に呼び出し**

| 項目 | 内容 |
|------|------|
| 価格 | 無料 / Pro版 $5（買い切り） |
| 状態 | ✅ 公開中 |
| バージョン | 2.1.0 |
| Chrome Web Store | [インストール](https://chromewebstore.google.com/detail/promptstash-ai-prompt-man/ocgkponbnolpgobllplcamfobolbjbcj) |

#### 概要
ChatGPT、Claude、Gemini、Perplexity、DeepSeekなど**19種類以上のAIサービス**に対応したプロンプト管理ツール。お気に入りのプロンプトを保存し、どのAIサービスでも即座に呼び出せます。

- **スラッシュコマンド**: 「/」入力で素早くプロンプトを呼び出し
- **19種類以上のAI対応**: ChatGPT、Claude、Gemini、Perplexity、DeepSeek、Grok など

#### 無料版
- プロンプト保存: 3件まで
- 変数テンプレート機能
- スラッシュコマンド
- 基本的なAIサービス対応（5種類）

#### Pro版の追加機能（$5 買い切り）
- プロンプト保存: 無制限
- すべてのAIサービス対応（19種類以上）
- プロンプトチェーン機能
- タグ・お気に入り機能
- 使用統計・バージョン履歴

#### こんな人におすすめ
- 複数のAIサービスを使い分けている人
- 繰り返し使うプロンプトがある人

### 7. 物件スカウター（購入版）

**不動産購入の意思決定をデータでサポート**

| 項目 | 内容 |
|------|------|
| 価格 | 無料 |
| 状態 | 🔧 提出準備中 |

#### 概要
物件スカウター（購入版）は、SUUMOの購入物件ページで坪単価・㎡単価を自動計算し、物件カードに表示するChrome拡張機能です。

数千万円の買い物だからこそ、感覚ではなくデータで判断したい。そんな方のために、築年数の色分けハイライト、お気に入り保存、比較表、価格履歴トラッキングなど、効率的な物件探しをサポートする機能を搭載しました。

完全無料で利用でき、データは全てブラウザ内にローカル保存。プライバシーも安心です。

#### 主な機能
- **坪単価・㎡単価の自動計算**: 物件カードに自動表示
- **築年数ハイライト**: 10年以内（緑）、20年以内（黄）、30年超（赤）で色分け
- **お気に入り保存**: 気になる物件を無制限に保存
- **比較表**: 複数物件を並べて検討
- **価格履歴トラッキング**: 物件の価格変動を記録
- **CSV/Excelエクスポート**: データを出力して家族と共有
- **ハザードマップ連携**: ワンクリックで災害リスクを確認
- **完全ローカル保存**: データは外部送信なし

#### こんな人におすすめ
- 中古マンション・戸建ての購入を検討中の方
- 坪単価で複数物件を比較したい方
- 割高な物件を避けたい方
- 物件情報を整理して家族と共有したい方
- 効率的に物件探しを進めたい方
